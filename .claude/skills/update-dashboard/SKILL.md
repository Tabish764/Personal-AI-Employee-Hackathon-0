# Update Dashboard Skill

## What to do:
1. Count files in /Needs_Action folder
2. Count files in /Done folder
3. Get current timestamp
4. Update Dashboard.md with:
   - Current timestamp in "Last Updated"
   - File count in "Files Pending"
   - List last 3 files in "Recent Activity"
5. Save the file

## Example output format:
Last Updated: 2026-01-10 11:30 PM
Files Pending: 2
Recent Activity:
- FILE_test.pdf detected at 10:26 PM
